What's Included in a Basement Development

A full basement development includes framing all walls and soffits, spray foam or batt insulation, vapour barrier, drywall, mudding and taping, priming, basic electrical rough-in coordination, and flooring. We coordinate with licensed electricians and plumbers where required.

Egress Windows

If your basement will include a bedroom, Regina's building code requires an egress window of minimum size for safe exit. We handle egress window installation including concrete cutting and proper waterproofing.

All basement development in Regina requires a building permit. We prepare and submit the permit application, coordinate inspections, and ensure your project meets current National Building Code requirements.
